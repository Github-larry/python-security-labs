Key Concepts Learned:

1. Python variables store references to objects
2. id() reveals object identity
3. Immutable objects create new memory when changed
4. Mutable objects modify existing memory
5. Shared references can create unexpected behavior
6. Security implication: mutable shared objects may cause data corruption

Lesson 3 – Python Data Types

Integers: numeric operations
Strings: text and user input
Lists: ordered mutable collections
Tuples: immutable collections
Dictionaries: key-value mappings
Sets: unique value collections
